Learn How To Play Guitar




Playing guitar is rather easy if you compare it with other instruments like violin and flute, but it still has its challenges. To start learn some easy chords like Am, Em ,C and G which are found in almost every song. After that practice transitions from one chord to the other. For example try changing from Am to C and them from Em to G and try to improve these transitions. There are abunch of videos to watch so you can learn how to play guitar but this one is my favourite.




After you practice transitions and learn these basic chords you can start to play some songs. Ive put 2 songs(riptide and back to friends) that are rather easy and fun to play, so you can start learning some. If you like to learn some more you can search on youtube or on google to find the chords.
